# 🍉 瓜了个瓜 - 网页版消除游戏

> 一款多层堆叠卡片消除游戏，纯 HTML/CSS/JS 开发，无需任何依赖。

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Tech](https://img.shields.io/badge/tech-HTML5%20%2B%20CSS3%20%2B%20JavaScript-green.svg)

## 🎮 游戏玩法

- **目标**：清空所有卡片即可获胜
- **规则**：
  - 点击未被完全覆盖的卡片，将其移入底部槽位
  - 槽位中收集到 3 张相同图案的卡片会自动消除
  - 槽位最多容纳 7 张卡片
  - 槽位满了且无法消除时游戏失败

## ✨ 特性

- **5 个递进难度关卡**
  - 第 1 关：12 张卡片，4 种图案，2 层
  - 第 2 关：15 张卡片，5 种图案，3 层
  - 第 3 关：18 张卡片，6 种图案，3 层
  - 第 4 关：21 张卡片，7 种图案，4 层
  - 第 5 关：24 张卡片，8 种图案，5 层
- **9 种精美图案**：🍉 🌾 🪵 🔥 💧 ⭐ 🍎 🌸 🌻
- **道具系统**：洗牌 + 撤回
- **精美动画效果**：
  - 可点击卡片金色边框呼吸动画
  - 卡片消除时闪烁放大特效
  - 胜利/失败弹窗动画
- **响应式设计**：完美适配 PC 和手机
- **智能失败检测**：确保每种图案数量为 3 的倍数
