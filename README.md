# 🐏 羊了个羊 - 网页版消除游戏

> 一款类似《羊了个羊》的多层堆叠卡片消除游戏，纯 HTML/CSS/JS 开发，无需任何依赖。

## 🎮 游戏玩法

- **目标**：清空所有卡片即可获胜
- **规则**：
  - 点击未被完全覆盖的卡片，将其移入底部槽位
  - 槽位中收集到 3 张相同图案的卡片会自动消除
  - 槽位最多容纳 7 张卡片
  - 槽位满了且无法消除时游戏失败

## ✨ 特性

- **5 个递进难度关卡**
  - 第 1 关：12 张卡片，2 种图案，2 层
  - 第 2-5 关：21 张卡片，7 种图案，3-6 层
- **7 种精美图案**：🐏 🌾 🪵 🔥 💧 ⭐ 🍎
- **道具系统**：洗牌 + 撤回
- **精美动画效果**：
  - 可点击卡片金色边框呼吸动画
  - 卡片消除时闪烁放大特效
  - 胜利/失败弹窗动画
- **响应式设计**：完美适配 PC 和手机
- **智能失败检测**：确保每种图案数量为 3 的倍数

## 🚀 快速开始

### 直接打开

```bash
# 克隆项目后，直接在浏览器中打开
open game.html
# 或
start game.html
```

### 本地服务器

```bash
# Python
python -m http.server 8080

# Node.js
npx serve
```

然后访问 <http://localhost:8080/game.html>
